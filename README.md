## Como baixar o projeto


```bash

  # Clonar o repositório
  $ git clone https://github.com/DevTeles/Calculadora-React.git

  # Entrar no diretório
  $ cd Calculadora-React

  # Instalar as dependências
  $ yarn

  # Iniciar o projeto
  $ yarn start

```
